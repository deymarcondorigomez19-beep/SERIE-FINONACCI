🌿 Contexto del Problema

¿Alguna vez contaste los pétalos de una margarita? Casi siempre encontrarás 13, 21 o 34 pétalos, nunca un número al azar. Esto no es casualidad: la naturaleza utiliza la Serie de Fibonacci para organizar el crecimiento de flores, hojas y semillas de la manera más eficiente posible.

Este fenómeno se llama filotaxia y describe cómo cada nuevo pétalo, hoja o semilla crece en una posición que maximiza el espacio disponible y la exposición a la luz solar. El resultado es una espiral perfecta que sigue exactamente los números de Fibonacci.

🧮 Explicación del Algoritmo

La Serie de Fibonacci es una secuencia donde cada número es la suma de los dos anteriores. Comienza con 0 y 1:

0+ 1→ 1→ 2→ 3→ 5→ 8→ 13→ …

Aplicado a las flores: cada nivel de crecimiento representa un término de la secuencia. El número en ese nivel indica cuántos pétalos o ramas podría tener la planta en esa etapa, siguiendo el patrón que la naturaleza ya utiliza.

El algoritmo usa tres variables simples (a, b, c) y un ciclo while para calcular cada término sin necesidad de vectores ni funciones complejas.

// Función principal que se ejecuta al presionar el botón
function calcularFibonacci() {

    // 1. Capturamos el valor del input usando getElementById

    var divResultado = document.getElementById('resultado');

    // 2. Validamos que el valor sea un número entre 1 y 30

        divResultado.innerHTML =
            "<span class='error-msg'>⚠ Ingresa un número entre 1 y 30.</span>";
        return;
    }

    // 3. Iniciamos las tres variables de la serie:
    //    a = término anterior (empieza en 0)
    //    b = término actual  (empieza en 1)
    //    c = variable temporal para el intercambio
    var a = 0;
    var b = 1;
    var c;

    // 4. Variable para construir el HTML del resultado
    var html = "<h3>Secuencia de " + niveles + " niveles:</h3>";
    html = html + "<div class='fib-grid'>";

    // 5. Ciclo while: se repite hasta cubrir todos los niveles
    var i = 1;
    while (i <= niveles) {

        // Mostramos el nivel y su valor Fibonacci

            "<div class='fib-item'>" +

        // Calculamos el siguiente término: c = a + b
        c = a + b;
        // Avanzamos: a toma el valor de b, b toma el valor de c
        a = b;
        b = c;

        i = i + 1; // Incrementamos el contador del nivel
    }

    html = html + "</div>";

    // 6. Renderizamos el resultado en el div de la página
    divResultado.innerHTML = html;
}

🌺 Conclusión

La Serie de Fibonacci no es solo un ejercicio matemático: es el código que usa la naturaleza para construir flores perfectas. Al programar este simulador, comprobamos que con solo tres variables y un ciclo, podemos reproducir un patrón que existe en millones de flores alrededor del mundo.

Esto demuestra que las matemáticas y la programación son herramientas poderosas para entender y modelar fenómenos naturales, convirtiendo ecuaciones abstractas en respuestas concretas y visuales.
